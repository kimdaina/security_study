ssh

## Level 1

사용한 명령어
- cd 
- cat

배운것

cd<br>
-> 현재작업 중인 폴더 위치 이동

\*<br>
cat - 명령어를 사용하면 막힘
' - ' 이거는 표준 입력(stdin)을 사용해라 라는 뜻이라서,
파일을 읽는게 아니라 키보드로 입력되는 내용을 읽어라라는 뜻이 됨
그러므로 경로를 표시하는 ./파일이름 와 같이 사용해서 현재 디렉토리에 있는 파일이름을 가진 파일을 연다라는 뜻이됨
./- 를 사용하면 -를 더이상 특별한 의미로 사용하지 않고, 경로로 인식함
<br><br>

## Level 2

사용한 명령어

- cat

\*<br>
만약 파일 이름이 --space in this file-- 이렇게 공백이 많은 파일이라면
Level1에서처럼 -를 특별한 의미로 인식하지 않고 경로로 인식하기 위해 ./를 사용
또한 공백을 경로로 인식하지 않고 문자 그대로 인식하기 위해 \(역슬래쉬) 사용
./--space\ in\ this\ file-- 이런식으로
<br><br>

## Level 3

사용한 명령어
- cd
- ls (옵션:a)
- cat

배운것 
ls -a
-> ls 뒤에 -a를 넣으면 숨겨져있는 파일도 함께 출력한다


## Level 4

사용한 명령어
- cd
- ls (옵션:a,l)
- file
- cat

배운것
ls -l
-> 이 디렉토리 안에있는 파일이 가지고 있는 권한도 함께 출력
    // 파일의 권한, 소유자, 그룹, 크기, 수정 시간 등을 자세히 출력

file
-> 파일의 형식과 종류를 확인

*
file ./* 를 사용하면 디렉토리안에있는 모든 파일의 형식과 종류를 확인 하고 그중 text형태의 파일을 화인
혹은
file ./* | grep "text" 현재 디렉토리안에 있는 모든 파일의 형식중 text 형태을 가진 파일만 출력
